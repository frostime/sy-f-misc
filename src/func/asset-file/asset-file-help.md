提供与附件文件相关的若干功能。

**新建附件文件**

在编辑器中任意位置输入 `/ni`，即可创建一个空白附件文件。该附件将存储在 `asset/user` 子目录下，以便与其他附件区分。

创建时可选择文件类型，例如 `md`、`txt` 等。预定义的文件类型可在插件配置中设置。

你可以在 `<工作空间>/data/public/blank-files/` 目录下创建空白模板文件，如 `blank.docx`。创建 `.docx` 文件时将自动使用该模板。

若未找到对应模板文件，系统将自动生成一个空白文件。

注意：默认情况下，docx、xlsx、pptx、prg（Project Graph）等文件必须使用空白模板，否则无法创建新附件。

而 md、txt 等简单文件无需模板也可创建。

**查看所有附件文件**

点击顶部菜单栏，可查看所有附件的文件列表。
