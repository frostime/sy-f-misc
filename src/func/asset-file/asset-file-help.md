提供若干和附件文件相关的功能。

**新建附件文件**

在编辑器任何地方输入 `/ni`  可以创建一个空白的附件，改附件会被存储在 `asset/user` 子目录下，以便和其他附件做区分。

可以选择创建文件的类型，例如 `md` , `txt` 等；预定义的文件类型可以在插件配置中设置。

注意：对于 Office 文件，例如 docx ，你需要在 `<工作空间>/data/public/blank-files/` 目录下创建对应的空白模板文件，例如 `blank-word.docx`, `blank-excel.xlsx`, `blank-ppt.pptx`。

**查看所有附件文件**

点击顶栏菜单，可以查看所有附件的文件列表。
