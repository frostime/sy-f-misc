本模块提供了 SuperRef 和 动态数据库两个功能。着眼于打通思源数据库和块结构。

- SuperRef：对 Tana SuperTag 理念的粗浅模仿，本质是自动获取文档的所有反向链接，然后自动化加入到数据库中；从而打通「反链-数据库」
- 动态数据库：给一个数据库绑定一段 SQL (JS 亦可) 查询语句，根据 SQL 搜索结构动态构建数据库
